# ArtCube Protocol Metadata Schema
## Canonical Bitcoin Ordinal Architecture
Version 1.1

## Overview
ArtCube is a Bitcoin-native protocol for anchoring artwork identity, provenance, legal title reference, and declared intellectual property rights.

Bitcoin serves as the canonical registrar of:
- artwork identity
- provenance history
- declared title state
- declared intellectual property rights

Other chains (such as Solana) may provide programmability but must reference Bitcoin as the canonical source of truth.
