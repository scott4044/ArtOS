# ArtOS MVP Scope

## Overview
ArtOS is a SaaS platform that allows artists, collectors, galleries, and institutions to tokenize **physical artwork on Bitcoin** using the **ArtCube Protocol**.

ArtOS provides a dashboard that enables users to:
- register accounts
- create artwork records
- generate Genesis Title Anchor metadata using the ArtCube Protocol
- upload supplemental files to Arweave
- inscribe canonical metadata onto Bitcoin Ordinals
- automatically generate a bridged token on Solana
- manage the asset lifecycle from a dashboard

The platform acts as **infrastructure for tokenized artwork**, not a marketplace.
